# 웹 구현용 화면 텍스트 / 더미 데이터

## 화면 1. 홈 / 온보딩

### 메인 카피
- 제목: 우리 아이 첫 100일
- 부제: 초등 1학년 학부모 지원 네비게이터
- 헤드라인: 입학 전후 100일, 지금 무엇을 먼저 해야 하는지 알려드립니다.
- 설명: 학교 정보, 방과후, 돌봄, 생활 적응 정보를 한 번에 정리하고, 우리 가정 상황에 맞는 행동 가이드를 제공합니다.

### 입력 필드
- 학교명
- 거주 지역
- 보호자 근무 형태
- 자녀 특성
- 현재 가장 큰 고민
- 현재 시점

### CTA
- 100일 플랜 만들기
- 데모로 바로 보기

### AI 요약 카드 예시
- 이번 주 우선순위
  1. 학교 앱과 알림장 확인 채널을 먼저 정리하세요.
  2. 첫 2주 하교 동선을 가능한 한 고정하세요.
  3. 방과후 신청 일정과 안내문을 이번 주 안에 확인하세요.
- 놓치기 쉬운 점
  - 등하교 방식이 자주 바뀌면 아이의 긴장이 커질 수 있습니다.
  - 공개 데이터와 실제 학교 공지는 다를 수 있으니 최종 확인이 필요합니다.

## 화면 2. 100일 타임라인

### 상단 텍스트
- 제목: 우리 아이 첫 100일 타임라인
- 진행 배지: 입학 첫 주 / D+5 / 100일 중 5일째
- 안내 문구: 지금 시점에서 중요한 일부터 순서대로 확인해보세요.

### 단계 탭
- 입학 준비
- 첫 2주
- 1개월 적응
- 돌봄 재조정
- 100일 회고

### 카드 예시
1. 학교 앱 및 알림장 채널 설정
   - 공지 누락을 줄이기 위해 안내 채널을 먼저 정리해야 합니다.
2. 첫 2주 하교 동선 리허설
   - 귀가 루틴이 안정되면 학교 적응 불안이 줄어듭니다.
   - 선배 팁: 첫 주에는 하교 장소와 귀가 방법을 자주 바꾸지 않는 것이 도움이 됐어요.
3. 방과후 신청 일정 확인
   - 운영 정보와 실제 신청 일정이 다를 수 있으므로 학교 공지를 함께 확인해야 합니다.

## 화면 3. 돌봄·방과후 추천 결과

### 상단 텍스트
- 제목: 돌봄·방과후 추천 결과
- 보조 문구: 우리 가정 상황에 맞는 선택지를 비교해보세요.

### 추천 카드 예시
1. 학교 방과후 기반 적응형 조합
   - 학교 안에서 이어지는 활동을 우선 활용해 이동 부담과 초기 불안을 줄이는 조합입니다.
2. 방과후 + 가정 루틴 병행 조합
   - 방과후 참여 횟수를 무리하게 늘리지 않고 가정 내 루틴 정착을 병행하는 조합입니다.
3. 돌봄 공백 대응 우선 조합
   - 하교 후 공백이 큰 요일을 우선적으로 메우는 방향의 조합입니다.

### AI 설명 패널
- 현재 상황 요약
- 추천 이유 3가지
- 놓치기 쉬운 점
- 선배 학부모 팁 요약
