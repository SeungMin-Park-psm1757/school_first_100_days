# 우리 아이 첫 100일
초등 1학년 학부모 지원 네비게이터  
교육 공공데이터 + AI 행동 가이드 기반 웹 MVP

## 프로젝트 개요
이 프로젝트는 초등 1학년 학부모가 입학 전후 100일 동안 겪는 정보 탐색과 의사결정 부담을 줄이기 위한 웹 MVP입니다.

핵심은 다음 3가지입니다.
- 홈/온보딩 화면에서 사용자 정보를 입력받고 AI 요약을 보여줍니다.
- 100일 타임라인 화면에서 시기별 행동 가이드를 제공합니다.
- 돌봄·방과후 추천 결과 화면에서 판단 근거를 설명합니다.

## 주요 기능
- 사용자 입력 기반 100일 플랜 생성
- 시기별 타임라인 안내
- 돌봄·방과후 추천 결과 제시
- 선배 학부모 팁 요약

## 구현 전제
- 모바일 퍼스트 UI
- 한국어 UI
- Next.js + TypeScript + Tailwind CSS
- 작동형 데모 수준 MVP
- 공공데이터 기반 정보와 선배 학부모 팁 시각적 분리

## 범위 제한
- 로그인 없음
- DB 없음
- 실제 학교 신청 자동화 없음
- 실시간 좌석/모집 연동 없음
- 더미 데이터 기반 시연용 MVP

## 실행 방법(예시)
```bash
npm install
npm run dev
```
